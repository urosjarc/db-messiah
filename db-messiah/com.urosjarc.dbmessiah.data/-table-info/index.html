fun getInsertRowBuilder(): RowBuilder

Returns a RowBuilder object that is responsible for generating SQL strings and query values based on a list of columns. The method constructs the RowBuilder object by combining the foreign columns and other columns of the TableInfo instance. If the primary column has an auto-increment or auto-UUID property, the foreign columns and other columns are used as is. Otherwise, the primary column is positioned at the beginning of the list followed by the foreign columns and other columns.
